How This Page Works:

For every day in the week, the first paragraph contents are changed. The HTML color is changed and the image is changed. In this page, there is a switch function with a parameter of myDay. The "?" in the url link is the start of the url parameter and "day" is the key for this parameter with a value of the case. The day that is grabbed from the querystring is passed into the switch function to retrieve the coffee object.

In the switch function, there are 7 cases from 0-6 and each case has a variable today that equals the day in the week. Sunday is equal to 0 and Saturday is equal to 6. Each case also contains a coffee object with 6 property names (pic, alt, desc, day, color, name). Each property name has a property value. The value for pic is an image of a type of coffee. The value for alt is a description of the image. The alt is a replacement for the image for the visually impaired. The desc is the description for that coffee. The day can be set to anyday you state it as. The color is the color the html page turns into for each day. The name is the the name of the drink.

On the html document, there is a div with an id of coffee cup which is selected by getElementById to change the html content in this div. This content is changed by the coffeeTemplate function. The coffeeTemplate function contains the first paragraph mentioned earlier that is altered and returned for every day in the week. The color of the html is changed by using querySelector to target the html and change the background color using "objectName.propertyName"(coffee.color). There are 7 links listed under "specials for other days" which when clicked on, will show the contents(image, description etc.) for that day. This is done by changing the value of day which changes the case number in the switch function. If you right click and click inspect, in the console it prints the value of day in the url (case number) and the coffee object.
